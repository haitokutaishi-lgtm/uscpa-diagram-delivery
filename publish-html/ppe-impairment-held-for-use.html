USCPA FAR / Long-lived assets（ASC 360-10）— レイアウトは低価法図解と同型

固定資産の減損（保有継続）：二段テストを「比較の相手」で切り替える

ステップ1では割引しない将来キャッシュフローの合計と帳簿価額を比べる。ステップ2では時価（fair value）が登場して損失額を測る——この切り替えが試験の腰です。

受講生

減損ってDCF で割引くイメージが強くて……。ステップ1だけ割引しないの、いつも逆にそうとしてしまいます。

コーチ

その混乱は普通。今日のゴールは「どのステップで、何と何を比べるか」を即答すること。低価法で FIFO／LIFO を分けたように、ここではステップ1／2で比較相手を切り替えるイメージです。

まず覚える3つ

ここまで押さえれば、本文の細部は後から足せます

1
ステップ1：回収可能性（recoverability） 必須

帳簿価額（carrying amount）と、割引しない将来期待キャッシュフローの合計を比べる。帳簿価額の方が大きい→ 回収不能サイン → ステップ2へ。

2
ステップ2：測定（measurement） 必須

減損損失は帳簿価額 − 時価（fair value）。試験では時価が与えられることが多いです。

3
ステップ1で止まるパターンがある 推奨

帳簿価額 ≤ 割引しないCF合計なら減損なし（ステップ2に進まない）。いきなり fair value で損失を出さない。

用語を口語に直す

用語ボックス

  • Carrying amount（帳簿価額）：テスト時点でバランスシートに載っている残高のイメージ。
  • Undiscounted expected future cash flows：将来に期待されるキャッシュフローを割引せずに足し合わせたもの。ステップ1専用の相棒。
  • Fair value（時価）：ステップ2で損失を測るときの下敷きとして使われることが多い（問題が数値を渡す）。
たとえ①

ステップ1は「まだ割引という調味料を入れない下ごしらえ」。まず素の合計で回収できそうか粗く見るイメージ。

たとえ②

ステップ2は「いくら損したかを時価ベースで確定させる」工程。いきなりここから始めない。

たとえ③

低価法の二段構え（時価の組み立て → 原価と比較）に似て、減損もステップ1 → ステップ2と順番が固定です。

分岐：ステップ1の結果で「次に進むか」が決まる

ここが試験の分かれ道です

ステップ1 合格側

CV ≤ undiscounted CF 合計 → 減損なしで終了

ステップ1 不合格側

CV > undiscounted CF 合計 → ステップ2へ進む

受講生

impairment loss と聞いたら、反射で CV − FV を書きたくなるんですけど……。

コーチ

その式はステップ2の世界。ステップ1で止まるなら損失ゼロ。まずundiscounted の文字を見つけて、どのステップにいるか決めよう。

ステップ1：回収可能性テスト

held for use の長期資産の典型（教材レベル）

比較の型

carrying amount vs. sum of undiscounted expected future cash flows

ステップ1では割引しない

資産グループ単位など、設問が変われば単位も変わります。英文のscopeを最初に確認してください。

ステップ2：減損損失の測定

ステップ1を通過したあとだけ

Impairment loss ＝ Carrying amount − Fair value

fair value が問題で与えられない場合の扱いは別の設問条件に従う（ここでは骨格のみ）。

ステップ1：CV と undiscounted 合計
不合格ならステップ2へ
CV − Fair value ＝ Impairment loss

3パターン：ステップ1の見極め → 必要ならステップ2

オリジナルのミニ例です

数字は架空。パターン1・3はステップ1で止まる、パターン2だけステップ2へ、という景色の違いを見比べてください。

1 パターン1
CV $500,000
Undiscounted 合計 $520,000
減損なし（Step2 なし）
2 パターン2
CV $800,000
Undiscounted 合計 $720,000
Step1 不合格 → FV $610,000
Loss $190,000
3 パターン3
CV $300,000
Undiscounted 合計 $410,000
減損なし（Step2 なし）

緑は結論の一言。パターン2だけ赤い行がステップ1不合格の合図。

演習で見るポイント

問題文を読む順番の型

ここで見るポイント
  • held for use か、別カテゴリ（例：held for sale）か——ルールが別物になります
  • undiscounted の有無でステップ1のレールを確認
  • fair value が与えられるタイミング（ステップ2）
  • 資産グループの範囲の説明

（疑似）問題文の読み方

「保有継続の長期資産／まず CV と undiscounted CF 合計／ダメなら CV − FV」——この型が頭に入っていると、長文から迷子になりにくいです。

理解確認（オリジナルMC）

ストップ・選択肢は試験英語／解説は日本語。配信前は別コンテキストで ASC 360-10 を照合してください。

問題 1（計算）

Cove Co. tests a long-lived asset that is held for use. The asset’s carrying amount is $650,000. The total undiscounted expected future cash flows are estimated at $580,000. The asset’s fair value is $500,000. What impairment loss should Cove recognize under U.S. GAAP?

  • A. $0
  • B. $70,000
  • C. $150,000
  • D. $650,000
解説を開く

ステップ1：CV $650,000 と undiscounted 合計 $580,000。650,000 > 580,000 なので回収可能性テスト不合格。

ステップ2：Impairment loss ＝ $650,000 − $500,000 ＝ $150,000。

A はステップ1見落とし。B は $650,000 − $580,000 のような別比較のイメージ。D は帳簿価額そのもの。

正解 C。

問題 2（概念）

For a long-lived asset held for use, the recoverability test under U.S. GAAP compares the carrying amount to which amount?

  • A. Fair value
  • B. Present value of expected future cash flows discounted at a market rate
  • C. Sum of undiscounted expected future cash flows
  • D. Replacement cost
解説を開く

C が回収可能性テストの比較対象。割引しない期待キャッシュフローの合計。

A はステップ2側。B は割引が入りステップ1のレールと別物になりやすい。D は標準の比較相手ではない。

正解 C。

まとめ：受講生チェック

コーチ

今日の3点要約です。

  • ステップ1は割引しないCF合計とCV。
  • ステップ1不合格のときだけ、CV − FV で損失。
  • held for sale など別分類は別レール。

受講生

自分の言葉だと、「まず undiscounted で生死を見る。生きてたら終わり。ダメなら fair value で損失」ですね。低価法と同じく、紙にレールを書いてから数字に入ります。
