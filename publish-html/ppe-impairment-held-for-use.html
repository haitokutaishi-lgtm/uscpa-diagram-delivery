今回のテーマ：Long-lived Assets — held for use の 2段階 impairment test

USCPA FAR / PPE Impairment（ASC 360-10）

保有継続の固定資産減損—recoverability test と impairment loss の比較相手を切り替える

試験では、ステップ1で帳簿価額（carrying amount）と割引しない将来 CF 合計を比べ、不合格ならステップ2で fair value を使って損失額を測ります。

① よくある悩み

試験でよく出る問題文の例

Cove Co. tests a long-lived asset that is held for use. The asset's carrying amount is $650,000. The total undiscounted expected future cash flows are estimated at $580,000. The asset's fair value is $500,000. What impairment loss should Cove recognize under U.S. GAAP?
  • A. $0
  • B. $70,000
  • C. $150,000
  • D. $650,000

このあと、たくまがつまずきやすい3つの考え方の誤りを対話で整理します。誤解①〜③は、試験でよく見かける間違いの番号です（選択肢 A〜D の記号とは別です）。

たくま（受講生）

誤解①　ステップ1 から fair value で比べる

fair value が $500,000 と出ているので、最初から CV $650,000 と比べて $150,000 が損失では？ ステップ1 を飛ばしてよい気がします。

あおい先生（コーチ）

held for use では必ずステップ1 からです。ステップ1 の比較相手は割引しない CF 合計であり、fair value ではありません。ステップ1 不合格のあと、初めて fair value で損失を測ります。

たくま（受講生）

誤解②　CV − undiscounted CF ＝ impairment loss

$650,000 − $580,000 ＝ $70,000（選択肢 B）が損失額ではないですか？

あおい先生（コーチ）

$650,000 と $580,000 の差は、ステップ1 で「回収可能性テスト不合格」と判断する材料です。損失額の計算式ではありません。損失は CV − fair value です。

たくま（受講生）

誤解③　CF が $0 でなければ減損なし（選択肢 A）

将来 CF が $580,000 もあるので、まだ回収できるから impairment loss は $0 では？

あおい先生（コーチ）

ステップ1 は「CF がゼロか」ではなく、CV が undiscounted CF 合計を上回るかです。$650,000 > $580,000 なので不合格。A はステップ1 の見落としです。

② まず覚える3つ

held for use の長期資産（ASC 360-10）です。

  1. 1

    ステップ1：CV vs 割引しない CF 合計

    CV > undiscounted CF 合計 なら回収可能性テスト不合格。ここでは fair value は使いません。

  2. 2

    ステップ2：impairment loss ＝ CV − fair value

    ステップ1 不合格の場合のみ。fair value が与えられていればその数字を使います。

  3. 3

    割引 PV はステップ1 に使わない

    present value of discounted cash flows は、held for use の回収可能性テストの比較相手ではありません。

③ 会計の背景と仕訳

②の3つは、ここでの帳簿価額が回収見込みを超えたときの write-downに対応します。

なぜこの処理か（会計の背景）

保有継続の長期資産は、将来キャッシュフローで回収できない見込みがあれば、帳簿価額を回収可能な金額まで下げます。まず割引しない CF 合計で回収可能性を確認し、不合格なら fair value 等で損失額を測定します（ASC 360-10）。

処理の順序： ①CV と undiscounted CF 合計を比較（回収可能性テスト） → ②不合格なら impairment loss ＝ CV − fair value → ③資産の帳簿価額を減らす

典型仕訳（Cove Co 例）

減損認識時（loss $150,000）：

Dr. Impairment loss                              150,000

    Cr. Accumulated depreciation (or Asset)        150,000

ステップ1：$650,000 > $580,000 → 不合格。ステップ2：$650,000 − $500,000 ＝ $150,000（誤解②の $70,000 ではない）。

次の④ 計算の型は、この impairment loss $150,000 を求める手順です。

④ 計算の型

Impairment loss を求める4ステップです。

  1. 1資産が held for use か確認する（held for sale 等は別ルール）。
  2. 2CV > total undiscounted expected future cash flows なら回収可能性テスト不合格。
  3. 3不合格の場合、fair value（問題文の数字）を確認する。
  4. 4Impairment loss ＝ CV − fair value。合格なら loss ＝ $0。

計算例（Cove Co・上の試験問題と同じ数字）

Undiscounted CF（合計）$580,000Step 1：CV > CF → 不合格

⑤ 対話で整理

たくま：誤解①〜③のあと、2段階の切り替えを確認したいです。 あおい先生：Cove の例では Step 1 不合格 → loss $150,000 です。

たくま（受講生）

選択肢 D の $650,000 は、帳簿価額を全部失う、という意味ですか？

あおい先生（コーチ）

fair value が $500,000 あるので、資産はゼロになりません。loss は CV と FV の差だけです。D は帳簿価額そのものを loss と取り違えた誤答です。

たくま（受講生）

割引した present value が問題文に出た場合はどうしますか？

あおい先生（コーチ）

held for use のステップ1 には使いません。英文が discounted や present value と言っていても、recoverability test の比較相手は undiscounted 合計です（②の3点目）。

⑥ 理解確認MC

オリジナル問題です。ASC 360-10 の考え方に沿って作成しています。

問題 1（計算 — impairment loss）

Cove Co. tests a long-lived asset that is held for use. The asset's carrying amount is $650,000. The total undiscounted expected future cash flows are estimated at $580,000. The asset's fair value is $500,000. What impairment loss should Cove recognize under U.S. GAAP?
  • A. $0
  • B. $70,000
  • C. $150,000
  • D. $650,000
解説を開く

Step 1：$650,000 > $580,000 → 回収可能性テスト不合格。

Step 2：Impairment loss ＝ $650,000 − $500,000 ＝ $150,000。正解 C。

A＝誤解③。B＝誤解②。D＝CV そのもの。

問題 2（概念 — recoverability test）

For a long-lived asset held for use, the recoverability test under U.S. GAAP compares the carrying amount to which amount?
  • A. Fair value
  • B. Present value of expected future cash flows discounted at a market rate
  • C. Sum of undiscounted expected future cash flows
  • D. Replacement cost
解説を開く

C が回収可能性テストの比較対象です。正解 C（誤解①の回避）。

A は Step 2 側。B は割引が入り Step 1 と別物。D は標準の比較相手ではありません。

⑦ この図解のまとめ

  1. held for use の減損は2段階。ステップ1 と 2 で比較相手が変わる（③の背景）。
  2. ステップ1：CV vs undiscounted CF 合計。fair value は使わない（誤解①）。
  3. CV − undiscounted CF は損失額ではない。loss ＝ CV − fair value（誤解②）。
  4. CF がゼロでなくても、CV が CF 合計を上回れば不合格（誤解③）。
  5. 仕訳：Dr. Impairment loss / Cr. Accumulated depreciation（or Asset）。
  6. held for sale や資産グループの問題は、英文の追加事実で別ルールに分岐する。

オリジナル教材。第三者問題・商用テキストの転載なし。
