今回のテーマ：PP&E — 一括購入の按分（相対売価／鑑定売価）

USCPA FAR / PP&E（ASC 360-10）

バンドル購入：合計現金を相対公正価値で割る

土地・建物などを一括で買ったとき、購入価格全体を各資産の鑑定売価（又は見積売価）の比率で配分します。売主の帳簿価額や土地だけに全部載せるなどの shortcut は試験の落とし穴になりやすいです。

よくある悩み

たくま（受講生）

問題に売主の土地簿価が書いてあるので、それで土地だけ決めてしまう。

あおい先生（コーチ）

買い手の取得原価の按分は自分側で見た各資産の相対FV。売主簿価は無関係であることが大半。

  • 合計鑑定額を足して各資産の割合を出す。
  • 一括現金 × 割合 ＝ 各資産のcapitalized cost。

まず覚える3つ

相対売価法（relative sales value method）。

  1. 1

    分子

    その資産の鑑定売価。

  2. 2

    分母

    グループ全体の鑑定売価の合計。

  3. 3

    配分

    一括購入価額 ×（分子÷分母）。

計算の型（表）

数字はイメージ。設問の鑑定額と一括価格に置き換える。

悩みが解けるまで（対話）

先の「売主の簿価で土地だけ決めそうになる」という悩みに、会話で按分の芯を当てはめます。

受講生：数字は足し算なのに、どの金額を分子にするかで迷う。 コーチ：買い手が払った一括価格を、自分たちの鑑定売価の比率で割るだけ。

たくま（受講生）

問題に売主側の土地の帳簿価額が書いてある。土地の取得原価はそれを使う？

あおい先生（コーチ）

典型は使わない。買い手の取得原価の按分は、土地・建物それぞれの鑑定（見積）売価の合計に対する割合で決める。売主の帳簿は背景説明か、別の落とし穴用のことが多い。

たくま（受講生）

仲介手数料みたいなのがついたら、按分の前に一括価格に足す？

あおい先生（コーチ）

設問が「取得原価に含める」と言っているものは、一括の取得コストに足してから比率按分するのが無難。書いていなければその問題の指示に従う。

理解確認MC

問題 1（土地の capitalized cost）

On January 2, Ridge paid $700,000 cash for land and a building as a combined purchase. Appraised values were Land $320,000 and Building $480,000. What amount should Ridge capitalize as land?
  • A. $280,000
  • B. $320,000
  • C. $350,000
  • D. $420,000
解説

合計鑑定 $800,000。土地の割合 = 320÷800 = 40%。

$700,000 × 40% = $280,000。正解 A。

問題 2（概念）

When several nonfinancial assets are acquired for a single lump-sum cash price, the cost allocated to each asset is generally based on:
  • A. Each asset’s carrying amount on the seller’s books
  • B. Relative fair values (for example, appraised values) of the assets acquired
  • C. Original historical cost of each asset
  • D. Equal allocation regardless of fair values
解説
B。

まとめ

  1. 分母は鑑定合計。
  2. 売主簿価は基本無視。
  3. 一括現金 × 比率で BS 計上額。
